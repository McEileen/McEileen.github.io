Class Methods and Instance Methods

When To Use What

August 16, 2015

Class methods are methods that are defined on a class. Class methods can be distinguished from other sorts of methods, such as instance methods, by their notation. They generally are written with the name of the class, a : : as separator, and then the method itself. For example, a class called Hamilton, with a method of calculate, would be written Hamilton::calculate. An instance method would use a # as a separator, like Hamilton#calculate.

Class methods are relevant because certain functions that are related to class cannot be accomplished by individual instances of a class. That’s why a class method sends a command to the class itself, rather than to one of the class’s instances.
