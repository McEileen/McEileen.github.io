Recursers regularly get together and work on interview problems during post-lunch prep, affectionately known as PLP, or "ploop". Today I ran post-lunch prep, in part because I wanted to choose our daily topic: bit manipulation! I didn't have much prior experience with bit manipulation, which is why I wanted a PLP focused on it.

Since we were an odd group, I paired/tripled with two Recursers, Jamie and Brandon. We worked on Leetcode Problem 693: Binary Number with Alternating Bits. Given the task of finding if a positive integer has alternating bits, we came up with the below solution. 

        
                class Solution:
                def hasAlternatingBits(self, n: int) -> bool:
                    last_value = n & 1
                    while n is not 0:
                        n = n >> 1
                        next_value = n & 1
                        if last_value == next_value:
                            return False
                        last_value = next_value
                    return True
